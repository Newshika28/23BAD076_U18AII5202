# Assignment 02 – Implementation of Data Visualization Techniques

## Objective
To explore data distribution and variability using advanced visualization techniques.

## Scenario
A startup analyzes e-commerce transaction data to understand customer spending behavior
and detect abnormal purchase patterns.

## Tools Used
- R Language
- ggplot2
- dplyr

## Visualizations Performed
- Histogram of transaction amounts
- Boxplot to detect outliers
- Heatmap of monthly sales intensity

## Files Included
- `code.R` – R program for visualization
- `output/` – Folder containing visualization outputs
  - `histogram.png`
  - `boxplot.png`
  - `heatmap.png`
- `README.md` – Assignment explanation

## Outcome
The visual analysis helps identify customer purchase patterns, detect high-value
transactions, and support data-driven business decisions.
