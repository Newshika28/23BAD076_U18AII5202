# Assignment 01 – EXPLORATORY DATA ANALYSIS AND VISUALIZATION

## Objective
To design appropriate data charts to analyze and compare academic performance indicators.

## Tools Used
- R Language
- ggplot2
- dplyr

## Description
This assignment analyzes internal assessment marks of first-year students
across five subjects using different visualization techniques.

## Files Included
- `code.R` – R program for data visualization
- `output.png` – Output charts (Bar, Line, Pie)
- `README.md` – Assignment explanation

## Outcome
The visualization helps identify low-performing subjects and supports
evidence-based academic decision making.
